HTML tags

1. <!DOCTYPE> tag: Doctype declaration

2. <html> tag: Encapsulates the entire HTML document

3. <head> tag: Presents information about the HTML

4. <title> tag: Sets the title of the page in browser title bar

5. <body> tag: Presents the content that appear on the webpage

6. <h1> to <h6>: Header lables for hierarchical arrange of headings

7. <p> tag: For storing paragraphs

8. <br> tag: Used to create a line break

9. Comment <!--> tag: Used to declare statements to be only human-readable and not parsed by HTML parser

10. <b> tag: Define bold text

11. <div> tag: Used to wrap a section of html content

12. <header> tag: Container for introductory content

13. <footer> tag: Container for information such as authorship, copyright, contact etc.

14. <main> tag: Specify main content of a document

15. <details> tag: Additional details that the client can view or hide

16. <summary> tag: Defines a visible heading for <deatils> element

17. <article> tag: Defines an article

18. <abbr> tag: Defines for the users what an abbreviation stands for when howered over

19. <address> tag: Used to wrap the section containing contact information

20. <span> tag: Similar to div, used for inline organization and styling

21. <script> tag: Used to add a script to our HTML

22. <noscript> tag: Alternative content if user has disable Javascript on their browser

23. <img> tag: Used to display an image on our browser

24. <ul> tag: Used to define an unordered list

25. <ol> tag: Used to define an ordered list

26. <li> tag: Used to define an item in unordered or ordered list

27. <a> tag: Used to define a hyperlink to another website or webpage on the same website

28. <nav> tag: Defines a set of navigation links

29. <button> tag: Used to place a clickable button on the webpage

30. <form> tag: Used to create a form for user input
